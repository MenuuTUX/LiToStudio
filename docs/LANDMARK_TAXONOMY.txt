3D OBJECT / LANDMARK LIST FOR MULTI-VIEW FEMALE MODEL
========================================================

Purpose:
This list defines visible 3D objects, clothing components, accessories, and useful semantic landmarks
for a SAM3 / multi-view LiTo conditioning pipeline.

Recommended naming style:
Use neutral academic labels such as "upper_torso_garment", "hip_strap_line", and "navel_piercing"
instead of casual or sensitive wording.
